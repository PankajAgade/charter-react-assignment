# Reward point - React Assignment

This project is a simple web application that calculates the reward points for a given amount.
Using React.js (react create app) and react-router-dom for routing.

project run command:
    npm start

Test command:
    npm test

File structure:
    - public/data : contains the customer and transactions data.
    - src/components : contains the react components.
    - src/pages : contains the react pages.
    - src/services : contains the api calls import from data json.
    - src/utils : contains the utility functions.
